# favorbot
让你的收藏夹不再吃灰

# description
可以收藏内容，ai总结，并推荐

1. 将想要搜藏的内容(链接)输入给bot
2. bot可以对内容总结并分类
3. bot可以根据已经收藏的内容进行推荐

# 开发方法
测试驱动开发、敏捷开发
